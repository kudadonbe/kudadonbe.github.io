މުދާ

މުދަލަކީ ﷲ ތަޢާލާގެ މިލްކެއް، އެ މުދާ މީސްތަކުންގެ އަތްމަތީގައިވަނީ އަމާނާތެއްގެ ގޮތުގައި، އެ ހޯދައިގަންނަންޖެހެނީ އެ އިލާހު ހުއްދަކުރެއްވި އުސޫލުތަކުގެ މަތިން، އެ ހޭދަކުރަންޖެހެނީ އެ އިލާހު ހުއްދަކުރެއްވި ގޮތުގެ މަތިންނެވެ. މިކަމަކީ ހިސާބު ބެއްލެވުމުގެ ދުވަހުގައި ސުވާލުކުރެވޭނެ ކަންކަމެވެ.

ހެއްދެވި ޚާލިޤެއް އެއްކައުވަންތަ އިލާހެއް ވާކަން ޤަބޫލުނުކުރާ ކާފިރުންގެ އުސޫލަކާ ގޮތްތަކުގައި މަސްލަޙަތު އެއްގޮތަކަށްވެސް ނުނުންނާނެއެވެ. އިޤްތިޞާދު ބައްޓަން ކުރަންޖެހޭނީ އުޑާ ބިމާ އެ ދެ މެދުގައިވާ ހާ ތަންތަނާ ތަކެތީގެ މިލްކުވެރި ފަރާތް އަމުރުކޮށް އެ ކަން ކުރަން އަންގަވާފައިވާ ގޮތަށެވެ.
