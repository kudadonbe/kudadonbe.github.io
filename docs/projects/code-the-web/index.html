ކޯޑް ދަ ވެބް

ކްލާސްރޫމުގައި ކިޔަވައިދެވޭނެ ވެބް ޑިވެލޮޕްމަންޓް ކޯހެއް — 14 ލެކްޗަރ ހިމެނޭ ގޮތަށް

2025 · ހިދުމަތުގައި ރިޕޯ

މިއީ ކްލާސްރޫމުގައި ކިޔަވައިދޭން ބޭނުންކުރެވޭނެ ވެބް ޑިވެލޮޕްމަންޓް ކޯހެކެވެ. މައިގަނޑު 14 ލެކްޗަރު އެކުލެވިގްންވާ މިކޯހުގައި އިތުރު 4 ލެކްޗަރު ހިމެނޭނެއެވެ.

ޕްރޮޖެކްޓް ތަކެއް ހަދަމުން ގޮސް އެންމެ ފަހުން ޕޯޓްފޯލިއޯ ސައިޓެއް ބިނާކުރުމަށް ދަސްކޮއްދެވޭނެއެވެ. ހުރިހާ މަސައްކަތެއް ވެސް ހުށައަޅައިފައި ވަނީ އެޗްޓީއެމްއެލް، ސީއެސްއެސް، އަދި ވެނީލާ ޖާވާސްކްރިޕްޓް ބޭނުންކޮށްގެންނެވެ. އެއްވެސް ފްރޭމްވޯކެއް ބޭނުންނުކޮށެވެ.

މަޤްޞަދަކީ ފްރޭމްވޯކެއް ބޭނުންކުރަން ފެށުމުގެ ކުރިން ބްރައުޒަރުން ކުރާ ކަންތައް ދަރިވަރުންނަށް ވިސްނައިދިނުމެވެ.
