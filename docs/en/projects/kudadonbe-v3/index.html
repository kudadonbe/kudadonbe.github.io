Kudadonbe v3

Rebuild of kudadonbe.mv on Astro — bilingual content site and developer portfolio.

Astro · Tailwind · MDX · Vue Repo

A rebuild of the personal site on Astro, after two previous tries on Angular. The motivation is to make writing the primary thing again — Dhivehi essays and poetry sit alongside developer work as first-class content, not buried under a project list.

Built with Astro for static generation, MDX for content with components, Tailwind for styling, and Vue for the few interactive bits. Deployed to GitHub Pages as a static build copied over to kudadonbe.github.io/docs/ by a small deploy script.
